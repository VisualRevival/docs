AWS

AWS Endpoints

Get download by ID

Gets a single download by ID

path Parameters
downloadId
required
string <uuid>

The ID of the download

query Parameters
filename
string
Example: filename=Filename123.mp4

The filename the browser should use when saving the file to the user's filesystem. If not provided, the last saved filename from AWS will be used.

Responses

AWS dev stage

https://obev9z9046.execute-api.us-west-2.amazonaws.com/download/{downloadId}

Response samples

  • 200
Content type
application/json
{
  • "id": "497f6eca-6276-4993-bfeb-53cbbbba6f08",
  • "url": "https://cloudfront.net/productid/downloadid.mp4?Expires=1234&Signature=ABCD"
}

Pack Processor Bundle

Schema for JSON file sent from Pack Processor to AWS for full bundle

id
required
integer <int64>

Bundle ID from Wordpress. This should always be present.

type
required
string
Value: "bundle"

Set to "bundle" to indicate bundle processing

productCategory
Array of strings

List of product categories across all products in bundle

productType
Array of strings

List of product types across all products in bundle

resolution
Array of arrays

List of resolutions across all products in bundle

frameRate
Array of arrays

List of frame rates across all products in bundle

audio
Array of arrays

List of audio types across all products in bundle

fileFormat
Array of arrays

List of file formats across all products in bundle

thumbnail
string

S3 Key of the uploaded thumbnail image for the bundle. Only required when creating a bundle.

previewHover
string

S3 Key of the uploaded hover preview image for the bundle. This is used in thumbnail listings. Only required when creating a bundle.

required
Array of objects (PackProcessorProductMediaItem)

List of product media for the product

required
Array of objects (PackProcessorBundleDownload)

List of downloads for the bundle

Array of objects (PackProcessorProduct)
